Sobre mí

¡Hola! Soy Andre, desarrollador Full Stack

Soy un ingeniero de sistemas con experiencia en el desarrollo de aplicaciones web. Tengo conocimientos de tecnologías en base de datos relacionales como MySQL, Oracle, PostgreSQL y SQLServer, y no relacionales como MongoDB.

En backend, me especializo en Java y Spring Boot empleando arquitecturas como Monolíticos y Microservicios. En frontend, utilizo herramientas como Angular, Bootstrap, Angular Material, HTML, CSS y JavaScript.
